## 머신러닝 AI 설계 시작

### 인공지능 (Artificial Intelligence, AI)

스스로 학습을 통해 스스로 판단하는 로직을 구동 (최적의 로직)

많은 양의 데이터 → 패턴을 찾는것

인공지능은 컴퓨터 시스템이 인간의 지능적인 행동을 모방할 수 있도록 하는 기술

AI는 학습, 추론, 문제 해결, 이해, 인식 및 언어 처리 등의 작업을 수행

Strong AI는 인간과 거의 비슷하거나 우월한 인공지능, Weak AI는 특정 문제를 해결하는 지능적 행동 AI

- 머신러닝 (Machine Learning): 데이터로부터 학습하고, 예측하거나 결정을 내릴 수 있는 알고리즘을 개발하는 기술입니다. 지도 학습, 비지도 학습, 강화 학습 등이 있습니다.
- 딥러닝 (Deep Learning): 신경망을 이용한 머신러닝의 한 종류로, 특히 대규모 데이터와 복잡한 패턴을 처리하는 데 유용합니다.
- 자연어 처리 (Natural Language Processing, NLP): 인간의 언어를 이해하고 생성하는 기술로, 번역, 챗봇, 음성 인식 등에 사용됩니다.

### 머신러닝 3가지 방식

**1. 지도학습 (Supervised Learning)**

입력값(데이터)과 결과값(레이블)을 함께 주고 학습을 시키는 방법으로, 과거의 데이터를 기반으로 미래의 이벤트를 예측할 때 유용하게 사용, 가장 많이 사용되는 학습 방식

예) 이전의 주식 데이터를 활용한 미래 예측 등

**2. 비지도 학습 (Unsupervised Learning)**

결과값 없이 입력값(데이터)만 주고 학습시키는 방법(결과값이 없음)으로, 데이터를 탐색하여 내부 구조와 유사한 속성들을 파악하여 분류하기 때문에 분류 기준들의 속성을 파악하는 데 효과적

클러스터링(군집)을 통하여 학습함

예) 쇼핑몰 고객 데이터를 분석하여 구매 패턴에 따라 고객을 그룹화, 이미지 분류 등.

**3. 강화학습 (Reinforcement Learning)**

결과값이 아닌, 어떤 일을 잘했을 때 보상을 주는 방식으로 어떠한 액션이 최선인지를 학습

게임, 네비게이션 등에서 활용되며 일정 시간 내에 예상되는 보상을 극대화시킬 수 있는 동작을 선택할 수 있도록 학습

예) 알파고 : 바둑게임을 스스로 두게 하여 승리시 보상을 하는 방식으로 학습

### 딥러닝 3가지 개념

**1. ANN(Artificial Neural Network)**

- 기존의 생물학적 신경망에서 영감을 얻어 만들어진 알고리즘
- 입력, 출력 사이 연결된 뉴런 계층으로 구성된 기본적인 인공 신경망
- 입력된 데이터에서 특징을 학습하고 분류 작업 진행
- 시간의 연속성이 있는 데이터를 처리하는 시계열 데이터 처리가 어려움

**2. ‍RNN(Recurrent Neural Network)**

- 이전 상태를 입력으로 받고 출력으로 데이터를 내보내면서 순환되는 신경망
- 시계열 데이터같이 입력 순서나 시간이 반영되어야 하는 문제들에 적합
- 문장 생성, 기계 번역 등의 작업에 적용됨
- 과거의 정보를 현재에서 활용할 수 있음

**3. CNN(Convolution Neural Network)**

- 입력 데이터에 커널을 적용하여 이미지의 특징을 추출
- 추출한 이미지의 특징은 다중 신경망을 다시 거쳐 요약되어 출력으로 사용됨
- 주로 이미지 처리에 사용되는 알고리즘
- 다양한 연산을 통해 공간적인 구조를 파악하고, 이미지의 특징을 활용해서 정확하게 예측함

### 빅데이터

많은 양의 데이터에서 목적(서비스)에 필요한 데이터를 추출 하는 기술

빅데이터는 크기, 속도, 다양성이 매우 큰 데이터 세트를 의미하며, 이러한 데이터를 분석하고 처리하여 유용한 정보를 추출하는 기술

빅데이터는 다음과 같은 특징 가짐

- **Volume (양)**: 데이터의 크기가 매우 큼
- **Velocity (속도)**: 데이터 생성 및 처리 속도가 매우 빠름
- **Variety (다양성)**: 데이터의 형태와 종류가 매우 다양
- **Veracity (정확성)**: 데이터의 정확성 및 품질 관리가 중요
- **Value (가치)**: 데이터로부터 유용한 가치를 추출하는 것이 핵심
