**웹(World Wide Web, WWW)**은 인터넷이라는 거대한 네트워크망 위에서 동작하는 **'정보 공유 서비스'**입니다. 흔히 인터넷과 웹을 같은 것으로 생각하기 쉽지만, 정확히는 **인터넷이 '도로'라면 웹은 그 도로 위를 달리는 '승용차'나 도로 옆의 '상점'**과 같습니다.

---

### 1. 웹의 핵심 개념

웹은 전 세계의 정보를 거미줄(Web)처럼 연결해 놓은 시스템입니다. 다음 세 가지 핵심 기술이 웹을 지탱합니다.

* **HTML (HyperText Markup Language):** 웹 페이지를 만드는 언어입니다. 텍스트뿐만 아니라 이미지, 영상 등을 담고 다른 페이지로 연결되는 '하이퍼링크'를 포함합니다.
* **HTTP (HyperText Transfer Protocol):** 웹 브라우저와 웹 서버가 정보를 주고받기 위한 통신 규약(약속)입니다.
* **URL (Uniform Resource Locator):** 인터넷상에 있는 수많은 정보 중 내가 원하는 정보가 어디에 있는지 알려주는 고유한 '주소'입니다.

### 2. 인터넷 vs 웹: 무엇이 다른가요?

많은 사람이 혼용하지만, 두 개념은 층위가 다릅니다.

| 구분 | 인터넷 (Internet) | 웹 (World Wide Web) |
| --- | --- | --- |
| **정의** | 전 세계 컴퓨터를 연결하는 **물리적 네트워크 인프라** | 인터넷을 통해 정보를 찾고 공유하는 **소프트웨어 서비스** |
| **비유** | 도로, 철도, 하드웨어 | 도로 위의 자동차, 기차 속 승객, 소프트웨어 |
| **포함 관계** | 더 넓은 개념 (웹, 이메일, FTP 등을 포함) | 인터넷 위에서 돌아가는 서비스 중 하나 |
| **시작** | 1960년대 (군사 목적 ARPANET) | 1989년 (팀 버너스 리 발명) |

### 3. 웹의 역사와 탄생

웹은 1989년 유럽 입자 물리학 연구소(CERN)의 **팀 버너스 리(Tim Berners-Lee)** 박사에 의해 처음 고안되었습니다. 당시 연구원들이 서로 다른 컴퓨터 환경 때문에 정보를 공유하기 힘들어하자, 어떤 컴퓨터에서든 정보를 쉽게 볼 수 있도록 '하이퍼텍스트' 개념을 도입한 것이 시작입니다.

* **1990년:** 최초의 웹 서버와 웹 브라우저 개발
* **1991년:** 전 세계에 웹 소스 공개 (누구나 무료로 사용 가능하게 됨)
* **오늘날:** 정적인 정보 전달(Web 1.0)을 넘어 사용자 참여형(Web 2.0), 인공지능 기반의 지능형 웹(Web 3.0)으로 진화 중
